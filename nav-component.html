Nav Component Reference

This file shows the navigation component in context and provides the copy-paste markup. The nav uses CSS-only dropdowns via :hover and :focus-within for keyboard accessibility, plus a JavaScript toggle for mobile.

How to Use

Copy the entire <nav class="main-nav">...</nav> block into any page. Include the styles from the <style> block above (or from page-template.html) and the JavaScript snippet at the bottom of this file.

Active State

Add class="active" to the current page's <a> tag in the nav to apply the crimson underline active indicator.

Accessibility Notes

Dropdowns respond to both :hover (mouse) and :focus-within (keyboard Tab navigation). The hamburger button toggles aria-expanded. Dropdown containers use role="menu" and items use role="menuitem".
